Prepare Yourself for Unix Ninjitsu

Choose Your OS

First, one must choose the right operating system

  • Linux is a good choice and is easy to run in a virtual machine on Windows to try out before making the switch
  • OS X has the Unix terminal built into the OS
  • If you are stuck on Windows, be sure to check out PuTTY to SSH into Unix machines.

Choose Your Sword

Your sword is your shell

  • Bash is the most common shell and the default on most Linux distros
  • Zsh is my shell of choice. It has more powerful tab completion and easier to customize than Bash

Use Screen

GNU Screen is a terminal multiplexer

  • Allows for multiple terminals at once
  • Saves session in case you lose connection
